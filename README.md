# Lujan Motors

Concesionario de vehículos desarrollado con Ember.js y Vite.

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm start  o npm run start
```

La aplicación estará disponible en http://localhost:4200

## Tests

```bash
npm test
```

Para ver los tests en el navegador, inicia el servidor y navega a http://localhost:4200/tests

## Limpiar caché

Si hay errores de build, ejecuta en PowerShell:

```powershell
Remove-Item -Recurse -Force .\tmp, .\dist, .\node_modules\.cache -ErrorAction SilentlyContinue
```
